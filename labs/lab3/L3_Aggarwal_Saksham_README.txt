# Inventory Management System

**A JavaScript-based inventory management system that allows users to add, remove, search, and categorize items, calculate total inventory value, and apply discounts. The system provides real-time inventory updates and interacts dynamically with the DOM.**

* *Date Created*: 13 Feb 2025
* *Last Modification Date*: 13 Feb 2025

## Authors

* Saksham Aggarwal (sk231042@dal.ca) - (Owner)

## Built With

* JavaScript, HTML5, CSS3

## Sources Used

None

## Features

✔️ Add new inventory items
✔️ Remove items from inventory
✔️ Search items by name or type
✔️ List all inventory items
✔️ Calculate total inventory value
✔️ Group items by category
✔️ Detect duplicate items
✔️ Apply discounts to all items
✔️ Real-time DOM updates
✔️ Error handling for invalid input
✔️ Console logging for debugging
